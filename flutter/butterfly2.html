Roaming Butterflies 2


This script displays randomly fluttering butterflies around the page. You can easily set how many of them you want.

A unique feature with this script is that the butterflies are NOT animated GIFs, they are seperate frames animated using javascript. All you need to do is specify the images and the order the images should be displayed. So, you can have all the same images, but the effect will be different depending on the order you specify.





Copy and paste the following script tag in the BODY section of your page:
Select all...





Then, edit the js file to set the number and order of the images you want to display.

You can also change a few other settings that determine how the script behaves.
You can also change the speed at which the images change (thereby increasing or decreasing animation speed).
